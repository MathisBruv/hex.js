# Hex.js

This module makes any string to hex.

### Installing

```
npm i hex.js
```

Example 

```js
const hex = require('hex.js');

console.log(hex.hex('Hi')); // Retruns 4869

console.log(hex.unHex('4869')); // Returns Hi
```
